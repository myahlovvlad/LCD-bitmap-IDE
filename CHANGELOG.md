# Changelog

## 0.1.1-public

- Prepared the repository for public open-source distribution.
- Replaced device-specific bundled material with a neutral universal LCD demo project.
- Added a professional product landing page under `docs/site/`.
- Added API/MCP connector documentation for local automation and agent-driven workflows.
- Expanded the in-app operation manual with an API/MCP automation chapter.
- Removed proprietary or device-specific artifacts from the public branch: decompiled firmware text, real screen catalogs, private prototypes, generated manuals and vendor-specific specifications.

## Notes

This public branch is intended to be published as an orphan branch so its git history does not contain previously committed local certificates or private development artifacts.
