# Screen DSL Examples

These examples are parser fixtures for Screen DSL V1.

- `minimal-screen.lcdscreen.json`: empty 128x64 screen.
- `minimal-screen.lcdscreen.yaml`: YAML equivalent of the minimal screen.
- `text-and-frame.lcdscreen.yaml`: rectangle plus localized text object.
- `resource-example.lcdscreen.yaml`: bitmap object with explicit resource ref.

All examples use `layoutMode: explicit`. They do not use table, grid, stack,
constraints, external resources, expressions or automatic layout.
